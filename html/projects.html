Project One ( Crack The Lock )

Play as a yellow ball and tap the screen when touching the red target. Collect keys and buy upgrades for a better score. Play on three Game Modes(Normal Mode, Hard Mode and Timer Mode). Get a greater highscore and eventually enough points to earn the code snippet of the lock from the main menu (Normal Snippet = 90, Hard Snippet = 45, Timer Snippet = 60). Get all the code snippets to unlock Mania Mode (Not a functional game mode currently).

Project Two ( Haunting Maze Escape )

Step into the world of a cave miner, finding themselves trapped in a undergound jail cell within a trailing cave system. The caves are a labyrith, filled with multiple paths, and an exit. In order to open the exit door you must find the button that opens it. On your path you can find numerous scattered coins. However, dont loose focus or else you will be found by a ghostly presence. Escape the haunting maze before its to late!
