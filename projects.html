Project Three

Net Link Chat App (Mar 10, 2025)

NetLink is an instant messaging app that allows users to communicate through the internet. Users can create an account, and add users to chat with. If you want to chat with someone else you can freely do so by searching for someone else, only if they are already registered in the database. The app has group chat functionality, as well as having a favorite chat. You can learn about the framework, tools, and challenges that I faced more through the button while making this project. As well as how to run it your self.

Project Five

Video Games Dashboard (Under progress)

A Front end web application that serves to help users visualize trends in video games. it can help users make informed decisions about their next download. It also help with exploration; popularity, top sellers, playtime, completion estimates, platform availability, genres, and much more. Ill be using Chart.js and Axios to call the API's. Im using Vite + React with bootstrap to help with styling. I will also implement a small backend server to create a proxy for each API. Still under the works, but the API I have researched are Steam Web API, RAWG Video Game Database API, and Internet Game Database (IGDB).
